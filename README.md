# Enfolder 
**Enfold** is an application designed to customize Windows 11 File Explorer folders in a practical and user-friendly way, aiming for the smoothest experience possible. While Enfold can be used on Windows 10, the application's primary focus is Windows 11.

## Download
You can download the .msi installer by [clicking here.](https://github.com/gabbdev/enfold/releases/latest)

## How to use?
After installing Enfold, open it for the first time. Once that's done, simply right-click the folder of your choice and set the style!
